# AGENTS.md — 26古诗文大赛工具

## 项目目标
- 本项目是本地网页版古诗文练习工具。
- 内容数据优先使用 Markdown 维护。
- 页面面向练习使用，优先保证离线可打开、检索清楚、题库方便核对。

## 目录约定
- `index.html`：本地网页入口。
- `styles.css`：页面样式。
- `app.js`：页面交互和临时内容索引。
- `content/`：每篇古诗文的 Markdown 内容库。

## 内容格式
每篇 Markdown 默认包含：
- `# 篇名`
- `## 作者`
- `## 出处`
- `## 原文`
- `## 注释`
- `## 文化常识`
- `## 对照翻译`
- `## 全文赏析`
- `## 题库`

题库保留原题和原答案；无法识别的图片内容标记为 `【待核对】`，不猜写。

## 开发规则
- 不引入构建工具，除非功能明显需要。
- 不添加外部依赖，优先使用原生 HTML/CSS/JavaScript。
- 页面应支持直接双击 `index.html` 打开。
- 修改后至少用浏览器可打开性或静态检查确认没有明显错误。
